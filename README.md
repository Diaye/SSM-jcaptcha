# 登录注册功能

使用SSM(Spring, Spring MVC, MyBatis)建立一个简单的框架，并实现简单的登录注册功能。

**IDEA** IDE，**tomcat** 服务器， **MySQL** 数据库。

## 特色

1. 注册页面使用 `jcaptcha` 实现了验证码功能，并使用AJAX技术实时验证。

2. 注册页面的邮箱输入完成，光标移开输入框后，使用AJAX技术到后台数据库进行查找，如果已经注册过则提示该邮箱已被注册。
